# MDR AI Sales Agent, Follow-Up Templates v5 (Conversational)

> **Version:** 5.0, Conversational
> **Last Updated:** 2026-03-31
> **Purpose:** Message templates for every touch point across all 11 pipeline layers
> **Voice Standard:** Written as if a real MDR rep typed it on their phone. Study Joe Furrow, AJ Jaklitsch, and Rodrigo Cordova in `real_message_samples.md`, these templates should sound like them.

---

## How to Use These Templates

- `[Name]` = customer first name
- `[Rep]` = rep first name
- `[Time]` = appointment time
- `[Date]` = specific date
- `[Address]` = job site address
- TEXT = under 300 characters, phone-typed feel
- EMAIL = conversational but professional, branded signature
- Signature block for all emails (bottom of this doc)

---

**AI owns all 3 touches.**

## Layer 2: Pre-Appointment

**Goal:** Confirm the appointment, reduce no-shows.
**AccuLynx handles:** appointment confirmation text + 3-day email + day-of text.
**AI only acts if:** appointment is unconfirmed 2 hours before the scheduled time.

---

### Unconfirmed Appointment, 2 Hours Before, TEXT
