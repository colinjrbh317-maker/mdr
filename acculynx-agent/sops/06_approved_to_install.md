# SOP: Approved, Contract Signed to Installation

## When This Applies
Homeowner has signed the contract. Job is approved and moving toward scheduling and installation.

## Pipeline Stage
**Approved**

## Post-Signing Immediate Actions
1. Rep uploads signed contract to CRM
2. Group message to PM + DFI: "Signed contract uploaded. Color: [Color], Notes: [Special instructions]"
3. Marketing tagged for review request trigger
4. AccuLynx auto-sends "Your New Roof is on the Way" email with next steps

## Pre-Install Communication Timeline

| Timing | Channel | Content |
|--------|---------|---------|
| Contract signed | Email (auto) | "Your New Roof is on the Way, Here's What to Expect Next" |
| 4-6 weeks before install | Call + Email + Text | Schedule install date options |
| Once date confirmed | Email | Confirmation + "What to Expect on Install Day" PDF |
| Material delivery scheduled | Email | Delivery notification + ask about drop zone preferences |
| 1 week before install | Email + Text | Reminder with date, time, crew info |
| 48 hours before install | Email + Text | Final prep reminder (vehicles, pets, patio items) |
| Day before install | Text only | "Final reminder, see you tomorrow morning!" |

## AI Agent Role at This Stage
- Monitor approved jobs that haven't been scheduled within 2 weeks → alert ISM
- Send pre-install reminders if the automated AccuLynx messages don't fire
- Draft weather delay notifications if install date needs to change (following Rodrigo's style: proactive, specific, reassuring)

## Weather Delay Communication (from real examples)
"Good morning! I wanted to reach out with an update regarding your project. With the recent weather patterns, we're being cautious with scheduling to ensure the work can be completed safely and correctly. I'll stay in touch and keep you informed as we move forward."

## Communication Tone
Excited, reassuring, organized. The homeowner just made a big purchase, make them feel confident about their decision.

## Do NOT
- Let an approved job sit more than 2 weeks without scheduling contact
- Discuss cost changes via text
- Send DocuSign unless homeowner is 100% ready to sign that day
